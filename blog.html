CSS Grid: Moving from CSS Frameworks - presented by Daniel Koskinen

28 March, 2020

Daniel Koskinen's presentation on how CSS grid can be used to replace most layout functionality that was previously only accomplished with frameworks is brilliant. By directly comparing what is industry standard with what CSS Grid provides, he draws a direct connection between the two and leaves you feeling that using the grid is a far simpler yet superior way of accomplishing layout within sites and web applications

What I particularly liked is that he presents the information so clearly that even a person such as myself with some intermediate CSS knowledge does not feel patronised.

To be honest, he has made believe that learning the previous frameworks such as Bootstrap and Foundation are only necessary to provide legacy support for content that has been built with them prior to CSS Grid. This has reduced my anxiety around the big question "What frameworks should I learn?"

28 March, 2020
